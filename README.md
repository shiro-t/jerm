# jerm
シリアル通信プログラム

かつて Candy (KANDA Toshihiro)氏が開発、公開されていた BSD 系UNIXのターミナル上で動作するシリアルポートとの通信プログラムである jerm になります。
以下サイトで公開されておりましたが、サイトがなくなり久しいため Github に転載をしたものです。  
http://bsddiary.net/jerm/

オリジナルのアーカイブは [original_archive](original_archive/)以下にあります。
また [src](src/) 以下には最新バージョンになる jerm-8096 のソースを展開しております。
コメント等が EUCjp だったため、UTF-8 への変換のみ行っております。

使い方については、[オリジナルの README](src/README)をご確認ください。
